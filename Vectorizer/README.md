## Download your vectorizer model 

You can download any feature extraction model for Malay language and put it here. <br>
Personally I use 3 models from huggingface which is:<br>

1. google/canine-c<br>
2. mesolitica/malaysian-debertav3-base<br>
3. intfloat/multilingual-e5-base<br>
4. base bert-model<br>

Ensure to change the path to the correct model when running the app.py 

## Huggingface Link

1. google/canine-c : https://huggingface.co/google/canine-c <br>
2. mesolitica/malaysian-debertav3-base : https://huggingface.co/mesolitica/malaysian-debertav3-base <br>
3. intfloat/multilingual-e5-base : https://huggingface.co/intfloat/multilingual-e5-base <br>
