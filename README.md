# Islamic-Chatbot

Welcome to the Islamic Chatbot project designed specifically for the domain of Hajj. This repository contains the code, resources, and information necessary to understand and utilize this chatbot effectively.

## Overview
The Islamic Chatbot for Hajj is a final year project aimed at providing a conversational interface tailored to assist pilgrims embarking on the Hajj journey. This chatbot serves as a virtual guide, offering information, guidance, and support related to various aspects of the Hajj pilgrimage in adherence to Islamic principles.

## How to use:
1. Install the dependencies in requirements.txt<br>
2. Run app.py file using streamlit run app.py
